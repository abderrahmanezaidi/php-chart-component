# PHP Chart Component

A reusable PHP component for generating charts.  
Designed to be easily integrated into PHP projects and extended with custom chart types.

## Features
- Modular chart component architecture
- Simple and consistent PHP API
- Supports multiple chart types
- Example usage included

## Requirements
- PHP 7.4+
- Composer
- Node.js (only if you use build/dev scripts)

## Installation

### Option 1: Clone the repository
```bash
git clone https://github.com/abderrahmanezaidi/php-chart-component.git
cd php-chart-component
composer install
